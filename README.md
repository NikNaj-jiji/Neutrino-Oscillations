# Neutrino-Oscillations
Neutrino oscillation project computing flavour–transition probabilities and showing that neutrinos must be massive. Includes PMNS mixing, CP-violation overview, and a coupled-pendula analogy to illustrate the physics.
# Repository Structure
neutrino_oscillations/
├── README.md                               # Project overview and instructions
├── report/
│   └── NeutrinoOscillations_Report.pdf     # Final written report
└── notebooks/
    └── coupled_pendula.ipynb               # Jupyter notebook: 2- and 3-pendula simulations

### **Folder Descriptions**

**`report/`**  
Contains the full written project report, including:
- motivation from the solar neutrino deficit  
- derivation of oscillation probabilities  
- role of mass eigenstates and flavour mixing  
- overview of the PMNS matrix and conceptual CP violation  
- physical analogy using two and three coupled pendula  

**`notebooks/`**  
Jupyter notebook implementing and visualising:
- equations of motion for two coupled pendula  
- equations of motion for three coupled pendula  
- numerical solutions using `scipy.integrate.odeint`  
- plots illustrating energy exchange and normal modes  

---

##  How to Use This Repository

1. Open the Jupyter notebook: notebooks/coupled_pendula.ipynb
2. Run all cells to:

solve coupled-pendulum equations

visualise oscillatory behaviour

explore how the analogy relates to neutrino flavour mixing

3. Read the PDF report for the full theoretical background.
